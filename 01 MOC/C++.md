# C++

## 学习目标
掌握 C++ 的核心语言特性（面向对象、泛型、内存管理）和标准库（STL 容器、算法、多线程），理解编译期和运行时的行为。

## 学习路径
1. 先理解 [[C++ 基础]] — 面向对象、内存管理、编译链接。
2. 再学习 [[STL 容器]]、[[迭代器]]。
3. 然后学习 [[C++ 线程基础与管理]]、[[C++ 线程同步]]。
