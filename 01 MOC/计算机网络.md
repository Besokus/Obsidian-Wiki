# 计算机网络

## 学习目标
理解网络协议如何影响服务通信、延迟、可靠性、安全性和故障排查。

## 核心问题
```text
1. 数据如何从一个进程发送到另一个进程？
2. TCP 如何提供可靠传输？
3. HTTP/HTTPS 如何承载应用层通信？
```

## 学习路径
1. 先理解 [[TCP]]、[[UDP]]、[[IP]]、[[DNS]]。
2. 再理解 [[三次握手]]、[[四次挥手]]、[[拥塞控制]]。
3. 然后学习 [[HTTP]]、[[HTTPS]]、[[TLS]]。
4. 最后结合 RPC、消息队列、分布式系统故障理解工程影响。

## 当前重点
这段时间优先推进什么？不要超过 3 项。
