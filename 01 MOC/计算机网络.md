# 计算机网络

## 学习目标
理解网络协议如何影响服务通信、延迟、可靠性、安全性和故障排查。

## 核心问题
```text
1. 数据如何从一个进程发送到另一个进程？
2. TCP 如何提供可靠传输？
3. HTTP/HTTPS 如何承载应用层通信？
```

## 学习路径
1. 先理解 [[TCP 协议]]（三次握手、可靠传输、流量控制）。
2. 再理解 [[HTTP 与 HTTPS]]（请求方式、状态码、缓存、TLS）。
3. 然后学习 [[DNS 与域名解析]]、[[认证机制]]。
4. 最后结合 [[Socket 编程与 TCP 黏包]]、[[输入URL后发生了什么]] 理解全流程。
