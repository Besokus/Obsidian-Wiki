# 操作系统

## 学习目标
理解进程、线程、内存、文件系统和 IO 模型如何影响后端程序的性能、正确性和稳定性。

## 核心问题
```text
1. 程序如何被调度执行？
2. 内存如何分配、隔离和回收？
3. IO 为什么会阻塞，如何提升并发能力？
```

## 学习路径
1. 先理解 [[进程、线程与协程]]、[[中断]]。
2. 再理解 [[虚拟内存]]、[[进程间通信]]。
3. 然后学习 [[IO多路复用与epoll]]、[[线程池]]。
4. 最后结合死锁、并发编程理解工程影响。

## 当前重点
这段时间优先推进什么？不要超过 3 项。
