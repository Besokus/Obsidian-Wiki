# 🌟解释一下虚拟内存

虚拟内存是计算机管理内存的一种技术，它允许操作系统将**硬盘空间用作临时的内存**使用。

虚拟内存是操作系统提供的一种内存管理机制，它通过给每个进程分配独立的虚拟地址空间，实现了内存隔离和扩展（将**硬盘空间作为临时内存使用**，使得可以运行更多程序）。
每个虚拟地址会通过页表映射到实际的物理内存页，当访问未加载的页时会触发缺页中断（**页面置换算法**），从磁盘中加载数据（将不频繁使用的数据暂时存放到硬盘上，以释放物理内存）。
这样既保证了系统安全，又提高了物理内存利用率。

**页面置换算法**：最佳置换（理想OPT）、**先进先出置换（FIFO）**、**最近最久未使用置换（LRU）**、**最近未使用**（Clock）、改进型 Clock



**为什么操作系统要使用虚拟内存？**

操作系统使用虚拟内存主要有几个目的：

1. 第一是实现**进程地址空间隔离**，每个进程都有独立的虚拟地址空间，提高系统安全性和稳定性。
2. 第二是通过分页机制实现按需加载，**实现内存扩展**，使程序看起来拥有比实际物理内存更大的空间。
3. 第三是**提高内存利用率**，只加载当前需要的页面。
4. 第四是实现内存保护，例如只读和可执行权限控制。
5. 虚拟内存还支持内存映射文件等高效的IO机制，将文件IO变成内存访问，效率更高
