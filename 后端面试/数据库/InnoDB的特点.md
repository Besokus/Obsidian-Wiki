# InnoDB的特点

InnoDB 是 MySQL 默认的存储引擎，主要特点包括：

1. 第一，**支持事务**，满足 ACID 特性，通过 undo log 和 redo log 实现事务的原子性和持久性。
2. 第二，**支持行级锁**，可以提高并发性能。
3. 第三，**支持 MVCC 多版本并发控制**，使读操作通常不需要加锁，从而提高并发读性能。
4. 第四，使用**聚簇索引结构**，数据和主键索引存储在同一棵 B+Tree 中，**主键查询效率较高**。
5. 第五，**支持崩溃恢复**，通过 redo log 在系统崩溃后恢复数据。
