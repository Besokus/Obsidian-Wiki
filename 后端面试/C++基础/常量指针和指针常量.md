# 常量指针和指针常量

| **常量指针** |      | `const Type *ptr;` | `const` 在 `*` **左边** | **指向的值**（数据）不能修改。 |
| ------------ | ---- | ------------------ | ----------------------- | ------------------------------ |
| **指针常量** |      | `Type *const ptr;` | `const` 在 `*` **右边** | **指针本身**（地址）不能修改。 |

**总结口诀**

判断 `const` 限制了什么，看 `const` 和 `*` 的相对位置：

1. **`const` 在 `*` 的左边**：限制的是**值**（指向的**数据**）。
   - `const Type *ptr;`
2. **`const` 在 `*` 的右边**：限制的是**指针**（**地址**）。
   - `Type *const ptr;`
