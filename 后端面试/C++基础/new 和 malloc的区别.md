# 🌟new 和 malloc的区别

1. 使用new操作符申请内存分配时无须指定内存块的大小，而malloc则需要显式地指出所需内存的尺寸并且进行强制类型转换（因为返回的是void*类型）

2. new / delete会调用**对象的构造函数/析构函数**以完成对象的构造/析构。而malloc则不会

3. malloc与free是C++/C语言的标准库函数, new/delete是C++的运算符

4. **`new `运算符** 从 C++ 抽象的**自由存储区**（底层也是堆）上为对象动态分配内存并进行构造；而 **`malloc` 函数** 从底层系统的**堆**上动态分配原始内存，不涉及对象的构造。
