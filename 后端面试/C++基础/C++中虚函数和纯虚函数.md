# C++中虚函数和纯虚函数

**虚函数 (Virtual Function)**

定义与作用

- **定义：** 在基类中使用 `virtual` 关键字声明的成员函数。
- **作用：** 实现 **动态多态**。它指示编译器，该函数的调用需要在程序**运行时**根据指针或引用的**实际对象类型**来决定，而不是在编译时根据指针或引用的类型来决定。
- **特性：** 虚函数在基类中==**必须有定义和实现**==，在派生类中可以被重写（Override），也可以不重写。

**实现机制**

- 虚函数依赖于 **虚函数表（vtable）** 和 **虚指针（vptr）** 实现。



**纯虚函数 (Pure Virtual Function)**

**定义与作用**

- **定义：** 虚函数在声明时后跟 `= 0`，并且不需要提供函数体。

**作用：** **强制**派生类提供该函数的具体实现，从而规范派生类的行为。

**特性：** 含有**纯虚函数的类**被称为 **抽象基类**，**不能**创建对象（不能被实例化）， 抽象基类只能作为**接口**或**规范**来使用
