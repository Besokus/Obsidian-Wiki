# C++中的nullptr与NULL的区别

**nullptr** ：nullptr是C++11引入的关键字，表示一种特殊的空指针类型，具体为`std::nullptr_t`线程安全类型，这种类型可以隐式**转换为任意的指针类型，但不能转换为整数类型**。

**NULL**：NULL是一个宏定义，通常定义为`0`或`(void*)0`，它的本质还是一个整数常量，可以隐式的**转换为指针类型**，但可能引发分歧。

| 特性       | nullptr         | NULL                  |
| ---------- | --------------- | --------------------- |
| 定义       | C++11新增关键字 | 宏，通常为表示整形为0 |
| 类型       | std::nulltpr_t  | 整数常量              |
| 类型安全性 | 强              | 弱                    |
| 转换为整数 | 不可以          | 可以                  |
| 推荐使用   | 是              | 不是                  |
