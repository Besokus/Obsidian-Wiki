# static关键字的作用

`static`关键字主要用于控制变量和函数的生命周期、作用域以及访问权限。

1. **修饰局部变量（函数内）**

   **特点：**

   - 生命周期：**程序整个运行期间（静态存储区）**
   - 作用域：**仍然在函数内部**
   - 初始化：**只执行一次**

2. 修饰**全局变量或函数**:

   当static用于修饰全局变量或函数时，限制了这些变量或函数的作用域，它们**只能在定义它们的文件内部访问**。有助于避免在不同文件之间的命名冲突。

   - 生命周期：**程序整个运行期间**
   - 作用域：**整个程序内**

3. **static 用在类中**

   **(1) static 成员变量**

   - **所有对象共享同一份数据**（不属于任何一个对象属于类本身）
   - **类内声明，必须在类外初始化**
   - **访问方式**：类名::变量名  或  对象.变量名

   **(2) static 成员函数**

- **不依赖对象实例**，没有隐式的this指针
- **只能访问static成员**，不能访问普通成员变量/普通成员函数（因为没有this）
