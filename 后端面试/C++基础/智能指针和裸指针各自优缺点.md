# 智能指针和裸指针各自优缺点

裸指针是普通指针，需要程序员手动管理内存，如果管理不当容易产生内存泄漏、悬空指针等问题。

智能指针是 C++ 提供的 RAII 封装类，可以**自动管理对象生命周期**，例如 `unique_ptr` 表示独占所有权，`shared_ptr` 通过引用计数实现共享所有权。

**智能指针的优点**是安全性高、异常安全，可以有效**减少内存管理错误**；缺点是可能带来**额外性能开销**，例如 `shared_ptr` 需要维护引用计数。

因此在一般应用开发中推荐使用智能指针，而在高性能或底层系统开发中，有时仍然会使用裸指针。
