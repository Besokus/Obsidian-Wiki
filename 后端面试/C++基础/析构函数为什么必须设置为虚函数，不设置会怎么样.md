# 析构函数为什么必须设置为虚函数，不设置会怎么样

当一个类作为基类，并且会通过**基类指针删除派生类对象**时，**析构函数必须声明为虚函数**，否则程序只会调用**基类的析构函数**，而不会调用子类的析构函数，会导致析构不完整，从而产生资源泄漏问题。



**构造函数可以为虚函数吗**

构造函数不能是虚函数。

因为虚函数的调用依赖于对象中的**虚函数表指针**（vptr），而在对象构造阶段，虚函数表还没有完全建立，此时对象本身还不完整，无法通过多态机制进行函数分发。

在对象构造过程中，vptr 是逐步初始化的：先指向基类的虚表，再切换到派生类的虚表。因此在构造阶段，对象的动态类型还不完整，多态机制无法正确工作。
