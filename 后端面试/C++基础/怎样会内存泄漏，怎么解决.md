# 怎样会内存泄漏，怎么解决

内存泄漏是指程序申请了堆内存但没有释放，并且程序已经失去了对这块内存的访问能力，从而导致这部分内存无法再被使用。
常见原因包括**忘记释放内存**、**指针被覆盖**导致地址丢失、异常导致释放逻辑未执行，以及**智能指针的循环引用**。

**解决方法**主要有三种：第一是保证 `new` 和 `delete` 成对出现；第二是使用 RAII 机制让资源随对象生命周期自动释放；第三是使用现代 C++ 的智能指针，例如 `unique_ptr` 和 `shared_ptr`，必要时使用 `weak_ptr` 来避免循环引用。
