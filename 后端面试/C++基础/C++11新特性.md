# C++11（最重要的一次升级）

1. **自动类型推导**

**auto**

编译器自动推导变量类型。

```c++
auto x = 10;
auto it = vec.begin();
```

作用：

- 减少冗长类型
- 提高泛型代码可读性



**2 .右值引用和移动语义**

**右值引用**

```c++
int&& a = 10;
```

作用：

- 区分左值 / 右值
- 支持资源转移

------

**移动语义**

通过移动而不是复制资源。

```c++
std::vector<int> a = {1,2,3};
std::vector<int> b = std::move(a);
```

优点：

- 避免深拷贝
- 提高性能



3. **Lambda 表达式**

可以定义匿名函数。



**4 .智能指针**

替代裸指针管理内存。

主要有：

```c++
unique_ptr
shared_ptr
weak_ptr
```



**5. 并发支持**

C++11 第一次引入标准线程库。

主要组件：

```
std::thread
std::mutex
std::condition_variable
std::atomic
std::future
```



6. **nullptr**

   替代 NULL。



**7. 范围 for循环**

简化容器遍历

```c++
for (auto& x : vec)
```
