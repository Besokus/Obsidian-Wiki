# 🌟在 C++ 中，浅拷贝和深拷贝的区别是什么？

浅拷贝拷贝对象的**指针值**，两个对象**共用同一块堆内存**，**当其中任意一个对象调用析构函数时，他们所维护的这个堆空间将不存在，容易造成双重释放空间。深拷贝拷贝的是指针所指向的**实际数据**，并为拷贝对象**重新分配内存，**相当于两个对象各自维护自己的堆栈空间互不干扰**

```C++
MyString(const MyString& other) {
    data = new char[strlen(other.data) + 1]; // 分配新内存
    strcpy(data, other.data); // 复制内容
}
```

通过 new 重新给 data 分配内存空间，而不是直接用赋值=代替（浅拷贝）

**C++ 默认构造函数、拷贝构造函数和赋值运算符都是浅拷贝。**
