# C++17

1. **结构化绑定**

```c++
std::pair<int,int> p{1,2};

auto [x,y] = p;
```



**2. if constexpr（模板编程神器）**

编译期条件判断。

```c++
if constexpr (std::is_integral<T>::value)
{
}
```

作用：

避免模板分支编译错误。



3. **std::any**

   可存储任意类型。

   ```
   std::any a = 10;
   ```



**4. std::optional**

表示 **可能为空的值**。

```c++
std::optional<int> value;
```



**5. 条件表达式中支持初始化语句**

```c++
if(int a = 0;)
switch(int a = 0)
```



**6. shared_mutex**

读写锁，性能提升。shared_mutex是c++的原生读写锁实现，有共享和独占两种锁模式，适用于并发高的读场景下，通过reader之前共享锁来提升性能。



7. **std::string_view (高性能必备)**

它只是一个“只读视图”（指针 + 长度），**不涉及内存分配和拷贝**。在处理高频字符串处理（如日志解析、协议解析）时，它可以大幅提升性能。

```c++
void process(std::string_view s) {
    // 无论传入 std::string 还是 char*，都不会产生拷贝
}
```
