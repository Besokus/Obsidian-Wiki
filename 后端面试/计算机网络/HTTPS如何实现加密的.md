# 🌟HTTPS如何实现加密的

**简要回复**

HTTPS = HTTP + **SSL/TLS**。
它通过 **对称加密** 保证**数据**传输安全，
**非对称加密** 保证**密钥**交换安全，
**数字证书** 保证服务器身份可靠。



**❓为什么要同时用对称 + 非对称加密？**

**非对称加密安全但慢**，对称加密快但密钥交换不安全；
结合两者：用非对称加密安全地传输对称密钥，之后用对称加密高效通信。

------

**❓如果证书被伪造怎么办？**

浏览器会验证证书链（CA 签名），
若签名无效或证书过期，会直接中断连接并报警。
