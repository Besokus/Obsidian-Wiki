# Http的状态码都有哪些，代表什么意思

HTTP 状态码分五类：
 1xx 信息、2xx 成功、3xx 重定向、4xx 客户端错误、5xx 服务端错误。

**❓1. 301 vs 302 区别**

301 是**永久重定向**，浏览器会缓存新地址；
302 是**临时重定向**，浏览器下次仍访问原地址。

------

**❓2. 401 vs 403 区别**（401：未登录；403：已登录但无权限。）

401：**未登录或 Token 失效**；
403：**已登录但无权限访问**。

------

**❓3. 502 vs 504 区别**（502：后端返回错误；504：后端没返回。）

502：网关收到**错误响应**（后端异常）；
504：网关**超时未收到响应**（后端没回）。

------

**❓4. 304 状态码与缓存机制**

304 表示资源未修改，客户端**可使用本地缓存；**
基于 `ETag` 或 `Last-Modified` 头判断**是否命中缓存。**
