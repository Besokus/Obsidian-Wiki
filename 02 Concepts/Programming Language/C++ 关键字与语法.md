# C++ 关键字与语法

## static

### 修饰局部变量
- 生命周期：程序整个运行期间（静态存储区）
- 作用域：仍在函数内部
- 初始化：只执行一次

### 修饰全局变量/函数
- 限制作用域为**当前文件内**，避免跨文件命名冲突

### static 成员（类中）
- **static 成员变量**：所有对象共享一份数据，类内声明、类外初始化
- **static 成员函数**：没有 `this` 指针，只能访问 static 成员

---

## extern

声明变量或函数定义在**其他文件中**，实现跨文件共享。

```cpp
// 文件A.cpp
int x = 10;
// 文件B.cpp
extern int x;  // 声明，链接到 A.cpp 中的定义
```

`extern "C"` 用于在 C++ 中调用 C 函数。

## nullptr vs NULL

| | nullptr | NULL |
|---|---|---|
| 定义 | C++11 关键字 | 宏，通常为 `0` |
| 类型 | `std::nullptr_t` | 整数常量 |
| 安全性 | 强（不可转整数） | 弱（可转整数） |
| 推荐 | ✓ | ✗ |

## const 与指针

| | 语法 | 含义 |
|---|---|---|
| 常量指针 | `const Type* ptr` | **指向的值**不能修改 |
| 指针常量 | `Type* const ptr` | **指针本身**不能修改 |

口诀：`const` 在 `*` 左边限制值，右边限制指针。

---

## 指针 vs 引用

| | 指针 | 引用 |
|---|---|---|
| 本质 | 存储地址的变量 | 变量的别名 |
| 重新赋值 | 可 | 不可 |
| 空值 | 可为 nullptr | 不能为空 |
| 解引用 | 需要 `*` | 不需要 |
| 多级 | 支持 | 不支持 |

### 使用场景

- **指针**：可能为空、需改变指向（如遍历链表）、动态内存分配
- **引用**：函数参数传递（避免拷贝）、运算符重载、链式调用

---

## 函数指针 vs 指针函数

**函数指针**：指向函数的指针，用于回调、动态调用。

```cpp
int (*ptr)(int, int) = &add;
int result = (*ptr)(3, 4);
```

**指针函数**：返回指针的函数。

```cpp
int* getPointer() { return new int(10); }
```

## define vs inline

| | define | inline |
|---|---|---|
| 阶段 | 预处理（文本替换） | 编译期（函数体插入） |
| 类型检查 | 无 | 有 |
| 安全 | 低 | 高 |

inline 只是对编译器的请求，复杂函数（循环、过多条件）可能被拒绝内联。

---

## define vs typedef

| | define | typedef |
|---|---|---|
| 阶段 | 预处理 | 编译/运行 |
| 类型检查 | 无 | 有 |
| 本质 | 文本替换 | 类型别名 |
