# C++ 现代特性

## 右值引用与移动语义

### 左值 vs 右值

| | 左值 | 右值 |
|---|---|---|
| 特征 | 有名字、能取地址、生命周期持久 | 无名字、不能取地址、临时一次性 |
| 例子 | `int a = 10;` 中的 `a` | 字面量 `10`、`a+b` 的结果 |

### 右值引用 `T&&`

专门引用右值的引用类型。唯一目的：**避免对临时对象的无意义拷贝**。

**移动语义**：通过移动构造函数和移动赋值运算符转移资源所有权，而不是深拷贝。被移动的对象处于"有效但未指定"状态。

**完美转发**：在模板中保留参数的左值/右值属性。

## 动态类型转换

| 转换 | 用途 |
|---|---|
| `static_cast` | 自然低风险转换（int ↔ float、父子类指针） |
| `dynamic_cast` | 多态基类到派生类的安全转换（检查安全性） |
| `const_cast` | 去除 `const` 属性 |
| `reinterpret_cast` | 不同类型指针/整数之间的强制转换 |

---

## 模板特化

为特定类型提供专门实现。

- **完全特化**：为某个具体类型提供完整实现
- **部分特化**：对部分模板参数特化（类模板支持，函数模板不支持）

## C++11 新特性（最重要的一次升级）

| 特性 | 说明 |
|---|---|
| `auto` | 自动类型推导 |
| 右值引用 + 移动语义 | 避免拷贝提升性能 |
| Lambda 表达式 | 匿名函数 |
| 智能指针 | `unique_ptr`、`shared_ptr`、`weak_ptr` |
| 并发支持 | `thread`、`mutex`、`condition_variable`、`atomic`、`future` |
| `nullptr` | 替代 NULL |
| 范围 for 循环 | `for (auto& x : vec)` |

## C++14 新特性

| 特性 | 说明 |
|---|---|
| Lambda 泛型参数 | `[](auto a, auto b){ return a + b; }` |
| 返回值类型推导 | `auto func() { return 10; }` |
| `std::make_unique` | 安全创建 unique_ptr |

## C++17 新特性

| 特性 | 说明 |
|---|---|
| 结构化绑定 | `auto [x, y] = pair;` |
| `if constexpr` | 编译期条件判断 |
| `std::any` | 可存储任意类型 |
| `std::optional` | 可能为空的值 |
| `std::shared_mutex` | 读写锁 |
| `std::string_view` | 字符串只读视图，零拷贝 |

## 静态链接 vs 动态链接

| | 静态链接 | 动态链接 |
|---|---|---|
| 时机 | 编译期打包进可执行文件 | 运行时加载外部共享库 |
| 文件体积 | 大 | 小 |
| 启动速度 | 快 | 稍慢（需加载库） |
| 更新 | 需重新编译 | 替换库文件即可 |
| 依赖 | 独立部署 | 依赖库版本一致性 |
