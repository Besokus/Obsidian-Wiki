# 中断

中断是操作系统处理**异步事件**的机制，允许 CPU 在执行程序时被"打断"，优先处理更紧急的任务。

## 分类

### 外部中断（硬件中断）

由外部设备产生：

- 键盘输入
- 定时器溢出
- I/O 完成

例如：敲击键盘时，键盘控制器产生中断信号，通知 CPU 读取输入。

### 内部中断（异常/陷阱）

由程序运行错误或特定指令触发：

- 除零错误
- 缺页中断（访问不存在的内存地址）
- 系统调用（用户程序请求内核服务）

## 中断的作用

- 提高系统实时性
- 实现 I/O 异步处理
- 支持多任务切换
- 实现系统调用入口
